# Rule Routing Skill

## Mục tiêu

- Chọn đúng nhóm rule trong `.ai/rules` theo ngữ cảnh công việc để tránh đọc thừa.

## Cách dùng

1. Luôn áp dụng `global-workflow-rule.md`.
2. Nếu task có BE, áp dụng thêm `backend-implementation-rule.md`.
3. Nếu task có FE, áp dụng thêm `frontend-implementation-rule.md`.
4. Nếu task fullstack, áp dụng cả BE và FE cùng với global rule.
