# Bảo mật

## 1. Mục tiêu tài liệu
- Tổng hợp các tính năng và cơ chế bảo mật của hệ thống.

## 2. Xác thực và phân quyền
- Cơ chế xác thực: <JWT/Session/OAuth/...>
- Mô hình phân quyền: <RBAC/ABAC/...>
- Nguyên tắc cấp/thu hồi quyền: <Mô tả>

## 3. Bảo mật dữ liệu
- Dữ liệu cần mã hóa khi lưu trữ.
- Dữ liệu cần bảo vệ khi truyền tải.
- Quy tắc ẩn/mask dữ liệu nhạy cảm trong log.

## 4. Bảo vệ API
- Rate limit.
- Kiểm tra input/validation.
- CORS/CSRF (nếu áp dụng).
- Chống injection và các tấn công phổ biến.

## 5. Quản lý bí mật và khóa
- Lưu trữ secret qua biến môi trường/vault.
- Quy trình xoay vòng khóa/secret.

## 6. Giám sát và ứng phó sự cố
- Audit log bảo mật.
- Cảnh báo bất thường.
- Quy trình xử lý incident.

## 7. Checklist hardening
- < > Tắt debug ở production.
- < > Bật HTTPS/TLS.
- < > Kiểm tra dependency có lỗ hổng.
